# Архитектура Vimax Core 0.1

Vimax Core — координатор. Технологии сыра, календарь салона и GPIO принадлежат модулям и коннекторам.

## Гарантии

1. Разговорная модель не вызывает оборудование и API напрямую.
2. Все действия типизированы.
3. Рискованные операции проходят Policy Engine.
4. План можно показать до выполнения.
5. Для production нужны idempotency keys и транзакционные бронирования.
6. Критические технологические функции работают локально без ИИ и Интернета.
